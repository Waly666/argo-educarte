Movimientos históricos (post-importación)

Matrícula con fecha anterior y recibos de migración en la ficha del alumno.

Asigne el permiso «Movimientos históricos de migración» en Configuración → Roles a quien operará la carga. Mientras esté desactivado, nadie puede usar la pestaña aunque tenga permiso.

1

Elija qué migrar

Cada cliente es distinto: solo certificados, alumnos con saldos, o todo.

2

Descargue la plantilla

Excel con solo las hojas que eligió, con ejemplos e instrucciones.

3

Valide el archivo

El sistema revisa fila por fila y le muestra los errores antes de tocar la base de datos.

4

Importe

Solo entran las filas válidas. Los duplicados se omiten y queda registro de cada lote.
